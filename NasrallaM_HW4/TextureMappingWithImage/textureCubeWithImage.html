Texture Mapping Demo

This program demonstrates:

  • Loading a gif or jpeg image as the texture. To get around the CORS browser policy, the image converted to a URI.(If you have a server running on your computer, you do not need this URI step.
  • How to set texture coordinates for a cube.
  • Using the sampler to interpolate texture coordinates over the triangles in the fragment shader.
  • In the fragment shader the color assigned to each face of the cube is blended with the color of the texture. Notice that the SIGGRAPH logo has a different color on each face. This color modulation can be turned off with the user interface below.
  • See the configureTexture function for settings. This uses the FLIP setting because of the organization of gif files.
